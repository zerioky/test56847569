# Interface complète Leia

Lancement :

```bash
cd Azip_py_projectleia_separed
python run_leia_ui.py
```

L'interface contient :
- chat réel relié à `LeiaLivingCore.respond()` ;
- panneau état vivant : confiance, méta-risque, tension, résonance, curiosité, sécurité, besoins internes ;
- inspection interne : résumé, impulsions, mémoire/digestion émotionnelle, JSON complet ;
- terminal UI ;
- boutons Self-test, Snapshot, Parole autonome, Forcer autonomie, Export JSON ;
- idle vivant activable/désactivable.

L'UI ne contient pas de réponses préécrites pour Leia. Elle affiche uniquement ce que le core produit.
